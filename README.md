#Infos Dashboard - Project

to up and run this project, after clone and access project folder, follow steps bellow

- Get dependencies
```shell
  $ yarn install
```


- Run development server
```shell
  $ yarn start
```



- Maybe you wanna build it, so run:
```shell
  $ yarn build
```
